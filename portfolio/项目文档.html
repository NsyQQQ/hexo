Redis - 实时数据

1. 用户会话

2. 房间信息 (实时)

3. 游戏状态 (实时)

4. 玩家房间状态 (实时)

5. 排行榜 (缓存)

数据同步策略

数据类型 存储 说明
用户信息 MySQL 持久化存储

牌局记录 MySQL 持久化存储
游戏状态 Redis 实时内存
房间状态 Redis 实时内存
排行榜 Redis 缓存，定期同步MySQL
