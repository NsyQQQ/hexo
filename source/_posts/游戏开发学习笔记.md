## 一、Egret简介

Egret（白鹭引擎）是一款成熟的H5游戏开发引擎，具有以下特点：

- **跨平台**：支持Web、iOS、Android
- **高性能**：基于TypeScript，编译效率高
- **生态完善**：丰富的图形特效和音频支持
- **商业成熟**：众多商业游戏采用

## 五、事件系统

### 5.1 常用事件

```typescript
// 触摸事件
let btn = new egret.Shape();
btn.graphics.beginFill(0x00ff00);
btn.graphics.drawRect(0, 0, 100, 50);
btn.graphics.endFill();

btn.touchEnabled = true;
btn.addEventListener(egret.TouchEvent.TOUCH_TAP, () => {
    console.log('按钮被点击');
}, this);

// 添加到舞台
this.addChild(btn);
```

### 5.2 事件类型

| 事件类型 | 说明 |
|----------|------|
| TOUCH_BEGIN | 触摸开始 |
| TOUCH_MOVE | 触摸移动 |
| TOUCH_END | 触摸结束 |
| TOUCH_TAP | 点击 |
| ADDED_TO_STAGE | 添加到舞台 |
| REMOVED_FROM_STAGE | 从舞台移除 |
| ENTER_FRAME | 每帧执行 |

## 六、动画与缓动

### 6.1 帧动画

```typescript
// 创建动画
let movieClip: egret.MovieClip = new egret.MovieClip();
movieClip.sheet = RES.getRes('hero_json');
movieClip.frameRate = 12;
movieClip.play();

// 播放
movieClip.gotoAndPlay('run', 0);

// 停止
movieClip.stop();
```

### 6.2 缓动动画

```typescript
// 使用Tween
egret.Tween.get(ball)
    .to({ x: 300, y: 200 }, 1000, egret.Ease.quadOut)
    .to({ x: 100, y: 100 }, 1000, egret.Ease.quadIn)
    .call(() => {
        console.log('动画完成');
    });
```

常用缓动函数：
- `linear` - 线性
- `quadOut` / `quadIn` - 二次缓动
- `cubicOut` / `cubicIn` - 三次缓动
- `bounceOut` - 弹跳效果

## 七、资源加载

```typescript

// manifest.json
{
    "resources": [
        { "name": "bg_jpg", "url": "assets/bg.jpg" },
        { "name": "hero_png", "url": "assets/hero.png" },

    let bg = RES.getRes('bg_jpg');
    let bitmap = new egret.Bitmap(bg);
    this.addChild(bitmap);
}, this);
RES.loadConfig('resource/default.res.json', 'resource/');
```

## 八、常用游戏组件

### 8.1 按钮

```typescript
class Button extends egret.DisplayObjectContainer {
    private bg: egret.Shape;
    private label: egret.TextField;
    
    constructor(text: string) {
        super();
        this.createButton(text);
    }
    
    private createButton(text: string): void {
        // 背景
        this.bg = new egret.Shape();
        this.bg.graphics.beginFill(0x4CAF50);
        this.bg.graphics.drawRoundRect(0, 0, 120, 40, 5);
        this.bg.graphics.endFill();
        this.addChild(this.bg);
        
        // 文字
        this.label = new egret.TextField();
        this.label.text = text;
        this.label.textColor = 0xffffff;
        this.label.size = 20;
        this.label.x = 60;
        this.label.y = 20;
        this.label.anchorX = 0.5;
        this.label.anchorY = 0.5;
        this.addChild(this.label);
        
        // 点击效果
        this.touchEnabled = true;
        this.addEventListener(egret.TouchEvent.TOUCH_BEGIN, () => {
            this.bg.alpha = 0.7;
        }, this);
        this.addEventListener(egret.TouchEvent.TOUCH_END, () => {
            this.bg.alpha = 1;
        }, this);
    }
}
```

### 8.2 进度条

```typescript
class ProgressBar extends egret.DisplayObjectContainer {
    private bar: egret.Shape;
    private bg: egret.Shape;
    private value: number = 0;
    private max: number = 100;
    
    constructor() {
        super();
        this.createBar();
    }
    
    private createBar(): void {
        // 背景
        this.bg = new egret.Shape();
        this.bg.graphics.beginFill(0xcccccc);
        this.bg.graphics.drawRect(0, 0, 200, 20);
        this.bg.graphics.endFill();
        this.addChild(this.bg);
        
        // 进度条
        this.bar = new egret.Shape();
        this.bar.graphics.beginFill(0x4CAF50);
        this.bar.graphics.drawRect(0, 0, 0, 20);
        this.bar.graphics.endFill();
        this.addChild(this.bar);
    }
    
    public setProgress(value: number): void {
        this.value = Math.min(value, this.max);
        this.bar.graphics.clear();
        this.bar.graphics.beginFill(0x4CAF50);
        this.bar.graphics.drawRect(0, 0, 200 * (this.value / this.max), 20);
        this.bar.graphics.endFill();
    }
}
```

## 九、碰撞检测

### 9.1 简单的矩形碰撞

```typescript
function hitTest(obj1: egret.DisplayObject, obj2: egret.DisplayObject): boolean {
    let rect1 = obj1.getBounds();
    let rect2 = obj2.getBounds();
    
    return rect1.intersects(rect2);
}
```

### 9.2 圆形碰撞

```typescript
function circleHitTest(x1: number, y1: number, r1: number, 
                      x2: number, y2: number, r2: number): boolean {
    let dx = x2 - x1;
    let dy = y2 - y1;
    let distance = Math.sqrt(dx * dx + dy * dy);
    
    return distance < (r1 + r2);
}
```
