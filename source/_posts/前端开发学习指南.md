## 一、HTML - 网页结构

HTML（HyperText Markup Language）是网页的基础结构，负责定义网页的内容和语义。

### 1.1 基本语法

```html
<!DOCTYPE html>
<html lang="zh-CN">
<head>
    <meta charset="UTF-8">
    <title>网页标题</title>
</head>
<body>
    <h1>一级标题</h1>
    <p>这是一个段落。</p>
    <a href="https://example.com">链接文本</a>
    <img src="图片地址" alt="图片描述">
</body>
</html>
```

### 1.2 常用标签

| 标签 | 说明 |
|------|------|
| `<h1>` - `<h6>` | 标题标签 |
| `<p>` | 段落标签 |
| `<a>` | 链接标签 |
| `<img>` | 图片标签 |
| `<div>` | 块级容器 |
| `<span>` | 行内容器 |
| `<ul>`、`<ol>` | 无序/有序列表 |
| `<table>` | 表格 |

## 二、CSS - 网页样式

CSS（Cascading Style Sheets）用于控制网页的外观和布局。

### 2.1 基本语法

```css
/* 选择器 { 属性: 值; } */
body {
    font-family: "Microsoft YaHei", sans-serif;
    background-color: #f5f5f5;
}

.container {
    width: 1200px;
    margin: 0 auto;
    padding: 20px;
}

.title {
    color: #333;
    font-size: 24px;
    text-align: center;
}
```

### 2.2 常用布局

**Flex布局：**
```css
.flex-container {
    display: flex;
    justify-content: center;
    align-items: center;
}
```

**Grid布局：**
```css
.grid-container {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}
```

### 2.3 选择器优先级

- ID选择器（`#id`）> 类选择器（`.class`）> 标签选择器（`tag`）
- 内联样式 > 内部样式 > 外部样式

## 三、JavaScript - 网页交互

JavaScript为网页添加交互功能，是前端开发的核心语言。

### 3.1 基本语法

```javascript
// 变量声明
let name = "张三";
const age = 25;

// 函数
function greet(name) {
    return `你好，${name}！`;
}

// 数组操作
const arr = [1, 2, 3, 4, 5];
arr.map(item => item * 2);  // [2, 4, 6, 8, 10]

// 对象
const person = {
    name: "李四",
    age: 30,
    sayHello() {
        console.log("你好！");
    }
};
```

### 3.2 DOM操作

```javascript
// 获取元素
const element = document.getElementById("myId");
const elements = document.querySelectorAll(".myClass");

// 修改内容
element.textContent = "新内容";
element.innerHTML = "<strong>HTML内容</strong>";

// 修改样式
element.style.color = "red";
element.classList.add("active");
```

### 3.3 事件处理

```javascript
// 点击事件
button.addEventListener("click", function() {
    alert("按钮被点击了！");
});

// 表单提交
form.addEventListener("submit", function(e) {
    e.preventDefault();
    // 处理表单数据
});
```

## 四、学习路线建议

### 阶段一：HTML + CSS（1-2周）
- 掌握HTML常用标签
- 学习CSS基础语法和选择器
- 了解盒模型和Flex布局
- 练习制作简单页面

### 阶段二：JavaScript基础（2-3周）
- 语法和数据类型
- 函数和作用域
- 数组和对象操作
- DOM基础操作

### 阶段三：进阶学习
- ES6+ 新特性
- 异步编程（Promise、async/await）
- 常用框架（Vue、React）
- 构建工具（Webpack、Vite）
