## 📋 项目概述

- **项目类型**: 微信小游戏
- **游戏类型**: 棋牌
- **前端**: Cocos2d
- **后端**: Node.js
- **数据库**: MySQL + Redis
- **部署**: Docker + Nginx

---

## 🏗️ 系统架构

```
┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐     ┌─────────────────┐
│   微信小程序    │────▶│   Nginx         │────▶│   Node.js       │────▶│   MySQL         │
│   (客户端)      │     │   (反向代理)     │     │   (游戏服务器)   │     │   (持久化存储)   │
└─────────────────┘     └─────────────────┘     └─────────────────┘     └─────────────────┘
                                                        │
                                                        ▼
                                                ┌─────────────────┐
                                                │   Redis         │
                                                │   (实时数据)     │
                                                └─────────────────┘
```

├── server/                 # Node.js 后端
│   ├── src/

## 🔧 技术栈

| 层级 | 技术 |
|------|------|
| 游戏引擎 | Cocos Creator 3.x |
| 编程语言 | TypeScript |
| 后端 | Node.js + Express + Socket.io |
| 通信协议 | WebSocket |
| 关系数据库 | MySQL 8.0 |
| 缓存数据库 | Redis 7.x |
| 部署 | Docker + Docker Compose |
| Web服务器 | Nginx |
